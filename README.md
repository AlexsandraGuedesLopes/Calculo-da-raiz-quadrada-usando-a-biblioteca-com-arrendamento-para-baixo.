# Calculo-da-raiz-quadrada-usando-a-biblioteca-com-arrendamento-para-baixo.
Calculo da raiz quadrada usando a biblioteca com arrendamento para baixo usando a condição floor.
import math
n1 = int(input("Digite um número:"))
raiz = math.sqrt(n1)
print("A raiz de {} é igual a {}".format(n1, math.floor(raiz)))
